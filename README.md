# Oliver Baptiste

## What I Do

I **design** and **develop** web sites and apps.
  * **UX design**
    * layout
    * color & typography
    * vector and raster graphics
    * user interface
    * animation
    * music & audio design
  * **Front-end development**
    * semantic markup with progressive enhancement
      * a11y & i18n
    * styling
    * scripting
    * browser testing
    * version control
    * deployment
